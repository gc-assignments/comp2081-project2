What users love about Sublime Text

Goto Anything

Use Goto Anything to open files with only a few keystrokes, and instantly jump to symbols, lines or words.

Triggered with ⌘P

Multiple Selection

Multiple selections allow you to interactively change many lines at once, rename variables with ease, and manipulate files faster than ever.

Try pressing ⇧⌘L to split the selection into lines and ⌘D to select the next occurrence of the selected word.

Command Palette

The Command Palette holds infrequently used functionality, like sorting, changing the syntax and changing the indentation settings.

Show the Command Palette with ⌘⇧P.

Customize Anything

Key Bindings, Menus, Snippets, Macros, Completions and more - just about everything in Sublime Text is customizable with simple JSON files. This system gives you flexibility as settings can be specified on a per-file type and per-project basis.

Sublime Text is available for OS X, Windows and Linux
